[SOFT] Reflashing the Nextbook Ares 8 tablet

tl;dr: One day our Nextbook Ares 8 tablets simply stopped working. This is how we fixed them.

The Nextbook Ares 8 tablet is a great device for its price range. We use ours for technical documentation and library access. They suddenly stopped working a while back, and based on XDA Forums cries for help, so did a lot of people's. This is how we fixed our tablets using resources posted out there and a Windows computer.

WARNING: Please note that all your data on the device will be wiped, and encryption will be disabled. We are NOT responsible for any damage to your tablet, your computer or any hardware involved in this tutorial.

Huge thanks go to XDA Forums users @vampirefo, @socialdesignconcepts and @greatbal for all their work on these machines.

Downloading and installing all the things

Let's start by installing all the software you will need on your computer.

  • Download and install the Intel Manufacturing Flash tool and the iSOC and ADB driver.
  • Follow the instructions to install the ADB and Fastboot tools systemwide.
  • Charge your tablet.
  • Download the Ares8_resize_partions.zip files from Vampirefo's Partition Ares 8 Tablet Linux and Windows script thread.
  • Extract the ZIP archive to your computer.

Everything should be ready. We can now start reflashing your tablet.

Reflashing the .EFI file

  • Open the unzipped ares8_resize_partions.zip folder.
  • In the unzipped folder, extract the esp.zip archive.
  • Open the Windows Command Prompt terminal and go to the unzipped esp folder using the cd command, all the way to the \EFI\Intel\ folder, where you will find the efilinux.efi.
  • Keep both VOLUME DOWN and VOLUME UP buttons pressed as you turn on the tablet. It should boot to a black screen reading Entering DNX mode. Waiting for fastboot command....
  • Connect your tablet to your computer with a USB cable.
  • From your terminal, type fastboot flash osloader efilinux.efi. Wait for the command to finish with an OKAY message on your screen.

Reflashing the bootloader

  • Use the cd command in your terminal to navigate back to the unzipped Ares8_resize_partitions folder.
  • Type fastboot boot droidboot.img. Wait for the command to finish with an OKAY message on your screen.
  • Turn off your tablet.
  • Keep VOLUME DOWN pressed as you turn on the tablet. It should boot into the Android bootloader (with the Android robot lying on its back).

Pat yourself on the back - you finished the first big step! Next up is rebuilding the partitions on your tablet and reflashing the operating system on it.

Repartitioning and reflashing

  • Browse to the unzipped Ares8_resize_partition folder.
  • In the folder, edit the fastboot_cmd.bat file to read all the fastboot commands.
  • From the terminal, type in all the commands, one at a time. Some of them will take time: be patient and let them finish with an OKAY message.
  • When you get to commands involving partition.tbl, replace the file name with stockpartition.tbl instead.
  • Type in the rest of the commands normally in the terminal, always waiting for OKAY.
  • Finally, use the VOLUME DOWN button to select RECOVERY MODE and use the POWER button to restart.

The tablet will now reboot into TWRP recovery. If it responds correctly to touch input, turn it off.

Reflashing the /system folder

  • Download the Ares8_stock_rom.zip file from the Partition Ares 8 Tablet Linux and Windows script.
  • Copy the Ares8_stock_rom.zip file on a MicroSD card and put it into your tablet.
  • Keep the VOLUME UP button pressed as you turn on your tablet to boot into TWRP recovery.
  • Go to INSTALL.
  • Select STORAGE, then MicroSD card.
  • Select the Ares8_stock_rom.zip file. Un-check the MD5 verifications option.
  • Swipe right to confirm flashing. You might get a warning about a missing partition path for /sdcard, but it shouldnâ€™t be a problem. The flashing will take a while - about 20 minutes on the tablets we restored. Do not interrupt it! If the screen goes to sleep, you can wake it up with a quick press on the POWER button.
  • When TWRP Recovery prompts you to wipe cache and DALVIK, swipe right to accept.
  • Finally, reboot the tablet.

The first boot will take at least 5 minutes, then walk you through the original setup for the tablet. Set up time zone, date and time and enjoy your reflashed tablet!
