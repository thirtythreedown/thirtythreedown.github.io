[SOFT] Shrinking Raspbian images in Linux

tl;dr: When you shrink your Raspbian images for storage, make sure to fold them carefully.

Copying your Pi image

In Windows, you can use Win32 Disk Imager's READ function instead of the WRITE function you usually use for writing images to your SD card.
In Linux, you can use Andrew Oaklay's instructions:

  • Find out where your SD card has been mounted - df -h
  • Unmount your SD card's partitions (it should have two) while leaving the SD card in the reader - sudo umount /dev/partition1 /dev/partition2
  • Copy the partitions to a .img file on your computer - sudo dcfldd if=/dev/sdcardmountingpoint of=imagename.img

Shrinking your Pi image

We're doing well! Time to shrink that image using a Linux computer (we use the latest Ubuntu LTS):

  • Download Drewsif's PiShrink script
  • Make the .sh script executable - sudo chmod +x pishrink.sh
  • Shrink the .img file you generated earlier - sudo ./pishrink.sh imagefile.img imagefile_shrunk.img

You're done!
