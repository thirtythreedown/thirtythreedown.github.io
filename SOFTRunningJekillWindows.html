[SOFT001] Running a Jekyll server on Windows

tl;dr: We've been curious about running a Jekyll server for development and found a lot of contradicting information across different sources. Here's what worked for us. We hope it helps.

First off, install the following software:

  • Git Bash
  • Ruby for Windows
  • This is it for Windows installs!

Next start Git Bash from the Start menu and:

  • Install jekyll and bundler - gem install jekyll bundler
  • Wait a while as Git Bash installs all the things!

Still in Git Bash, let's prep your website:

  • Browse to where you want your site folder on your computer - cd /path/to/whereyouwannawork
  • Create website - jekyll new nameofwebsite
  • Jekyll will build the site structure in the nameofwebsite folder.
  • Browse to the created folder - cd nameofwebsite
  • Start the Jekyll server from inside your website's folder - jekyll serve
  • Open a browser to view your site at http://127.0.0.1:4000.

You can stop the Jekyll server anytime with CTRL+C in Git Bash. Enjoy!
